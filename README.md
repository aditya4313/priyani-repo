# Priyani Kumari - Interactive Tech Portfolio

A premium, visually stunning, and highly interactive developer portfolio website showcasing my technical capabilities, education, work experience, and featured projects. Built to impress recruiters and tech leaders with clean design standards and interactive browser experiences.

🚀 **Live Preview Grid & Playable Pacman Game Included!**

---

## ✨ Features

- **Rich Dark Aesthetics & Custom Themes:** Glassmorphism card systems, harmonious neon gradient highlights (cyan, purple, and pink), and fluid dark/light mode toggles.
- **Dynamic Micro-Animations:**
  - Dynamic typewriter effect introducing my roles in software engineering.
  - Animated skill meters that expand dynamically when scrolled into view.
  - Sticky glassmorphic navigation header showing active sections on scroll.
- **Interactive Project Demos:**
  - **Playable Pacman Arcade Game:** A complete 2D arcade Pacman game developed from scratch on HTML5 Canvas. Supports arrow key navigation and responsive mobile touch swipe controls.
  - **AI Trip Planner Mockup:** A mini-app simulator generating customized day-wise itineraries dynamically.
  - **React Food App Mockup:** An interactive ordering simulator updating cart totals with animated feedback.
- **Academic & Experience Timelines:** Sleek, modern grids visualizing my current B.Tech in Computer Science journey at IITM (CGPA: 7.6), internship accomplishments, and hackathon wins.
- **Floating Particles Background:** A light, interactive custom canvas network connecting nodes based on cursor proximity and screen size.

---

## 🛠️ Tech Stack & Resources

- **Core:** Semantic HTML5, Vanilla CSS3 (Custom Variables, Flexbox/Grid, Keyframe Animations)
- **Scripting:** Vanilla ES6+ JavaScript (DOM APIs, Canvas API, IntersectionObserver, requestAnimationFrame)
- **Icons & Fonts:** FontAwesome 6.4.0 (CDN), Google Fonts (Plus Jakarta Sans & Space Grotesk)

## 💻 Local Quick Start

To preview the website locally on your computer, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aditya4313/priyani-repo.git
   cd priyani-repo
   ```

2. **Start a local development server:**
   - **Using Python (Recommended):**
     ```bash
     python3 -m http.server 8000
     ```
   - **Using Node (npx):**
     ```bash
     npx serve
     ```

3. **Open the browser:**
   Navigate to `http://localhost:8000` to view the live responsive portfolio!
